# Gerador SOAP — Emagrecimento — v24
- **Prescrição editável diretamente** na prévia (contenteditable).
- **Sem linhas divisórias** entre medicamentos (apenas uma linha em branco separando blocos).
- Impressão usa **exatamente** o que está na prévia editável.
- Receita horizontal em **2 vias**, layout idêntico ao v23, com **data formatada dd/mm/aaaa** nas vias.
- Botões de fonte **menores** (10 px / 12 px).

Como usar:
1) Selecione o medicamento principal (Wegovy/Mounjaro), clique em “Pré-visualizar principal” e/ou “Adicionar principal”.
2) Use “+ Adicionar Ondansetrona / PEG-Lax / Item avulso” para completar.
3) Edite livremente no bloco **Prescrição (prévia)**.
4) Clique “Salvar & Imprimir (2 páginas)”.
